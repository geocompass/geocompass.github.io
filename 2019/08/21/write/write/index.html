五、常见问题

5.1 部署不成功

  • 请严格按照 1.5 部署 流程提交一下你的code, 也许你的问题会解决。
  • 关于为什么要移动 .git文件 请查阅Hexo部署报错Spawn failed及解决方案
  • 部署成功后网站并不是最新的，请清楚一下浏览器缓存，再尝试。
  • 关于你的博客取用图片，你可以建立自己的GitHub图片供应仓库，例如GitHub图片供应库
  • 更多问题请提交Issues
